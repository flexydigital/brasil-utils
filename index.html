O que é o brasil-utils?

O brasil-utils é uma biblioteca PHP com classes utilitárias para validações e formatações comuns no Brasil, como:

  • CPF.
  • CNPJ.
  • RG.
  • PIS.
  • CEP.
  • Telefone Fixo.
  • Celular.
